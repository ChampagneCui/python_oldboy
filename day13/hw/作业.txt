主题：学员管理系统

需求：

用户角色，讲师＼学员， 用户登陆后根据角色不同，能做的事情不同，分别如下
讲师视图
　　管理班级，可创建班级，根据学员qq号把学员加入班级
　　可创建指定班级的上课纪录，注意一节上课纪录对应多条学员的上课纪录， 即每节课都有整班学员上， 为了纪录每位学员的学习成绩，需在创建每节上课纪录是，同时为这个班的每位学员创建一条上课纪录
　　为学员批改成绩， 一条一条的手动修改成绩

学员视图
提交作业
    查看作业成绩
    一个学员可以同时属于多个班级，就像报了Linux的同时也可以报名Python一样， 所以提交作业时需先选择班级，再选择具体上课的节数
    附加：学员可以查看自己的班级成绩排名


数据库模型：
讲师
    id
    name
    passwd

    例  id  name passwd
        1   alex 123456
        2   dashen 123456

学员
    id
    name
    passwd
    qq

    例 id name  passwd   qq
        1 leo   123456  11111111
        2 jack  123456  22222222

课程
    id
    name
    detail
    classroom_id
    例 id    name       detail       classroom_id
        1   python1     python基础1   1
        2   linux       linux介绍     2
        3   python2     python基础2   1

班级
    id
    name
    teacher_id

    例 id name  teacher_id
       1  python 1
       2  linux1  2

情况表
    course_id
    student_id
    attendance
    score

    例course_id   student_id  attendance score
        1           1           1           80
        1           2           0           0
